Question: Is chartJS a library or an api?

Answer:
Chart.js is a free open-source JavaScript library for data visualization, which supports 8 chart types: bar, line, area, pie (doughnut), bubble, radar, polar, and scatter.

Question: How do I run ChartJS on my web application?

Answer:
  1. In your main html page of your application, usually a index.html or main.html page, add the script tag in your header, like below: <head> <script src="https://cdn.jsdelivr.net/npm/chart.js@2.8.0"></script> </head>
  2. Add a canvas element in your main HTML page to render your graphical charts built using ChartJS like: <canvas id="myChart"></canvas>
